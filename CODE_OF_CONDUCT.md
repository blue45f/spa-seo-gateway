# Contributor Covenant Code of Conduct

## 우리의 약속

이 프로젝트의 메인테이너와 기여자는 모든 참여자에게 — 나이, 신체 사이즈, 장애, 인종, 성 정체성과 표현, 경험 수준, 국적, 외모, 인종, 종교, 성적 정체성과 지향에 관계없이 — 괴롭힘 없는 환경을 보장합니다.

## 우리의 기준

긍정적 환경에 기여하는 행동 예시:

- 환영적이고 포용적인 언어 사용
- 다른 관점과 경험 존중
- 건설적 비판 수용
- 커뮤니티에 가장 좋은 방향에 집중
- 다른 커뮤니티 구성원에 공감 표현

용납될 수 없는 행동 예시:

- 성적 언어나 이미지 사용, 원치 않는 성적 관심
- 트롤링, 모욕적/경멸적 댓글, 개인적 / 정치적 공격
- 공개적 또는 사적 괴롭힘
- 명시적 허가 없이 타인의 개인 정보(주소, 이메일 등) 공개
- 전문적 환경에서 부적절하다고 합리적으로 판단되는 행동

## 메인테이너 책임

메인테이너는 용납될 수 없는 행동에 대해 적절하고 공정한 시정 조치를 취할 책임이 있습니다 — 댓글 / 커밋 / 코드 / 위키 편집 / 이슈 / PR 의 제거, 편집, 거부 포함. 부적절하다고 판단되는 기여자를 일시 또는 영구 차단할 수 있습니다.

## 적용 범위

본 행동 강령은 프로젝트 공간(GitHub 저장소, 이슈, PR, 커뮤니케이션 채널) 과 개인이 공식적으로 프로젝트를 대표하는 공개 공간에 적용됩니다.

## 시행

학대, 괴롭힘, 기타 용납될 수 없는 행동 신고는 메인테이너 — `blue45f@gmail.com` — 에게 연락해 주세요. 모든 신고는 검토 후 상황에 적합한 응답이 이루어집니다. 메인테이너는 신고자의 기밀을 유지할 의무가 있습니다.
